# Plataforma de Streaming Personal

Este proyecto es una plataforma de streaming de películas y series desarrollada como parte de mi proyecto estudiantil. Es un espacio de prueba y desarrollo personal.

## Uso

- Abre el archivo `index.html` en tu navegador web para acceder a la interfaz de usuario.
- Utiliza los archivos en la carpeta `scripts/` para desarrollar la lógica de la plataforma.
- Los estilos están definidos en la carpeta `styles/`.

## Estructura del Proyecto

- `index.html`: Página principal de la plataforma.
- `styles/`: Carpeta con archivos CSS para estilos.
- `scripts/`: Carpeta con archivos JavaScript para funcionalidades.

## Notas Adicionales

- Este proyecto es exclusivamente para propósitos de aprendizaje y desarrollo personal.
- No se requiere instalación adicional más allá de abrir el archivo `index.html` en un navegador web.
